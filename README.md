## ELPL

Provides a simple interface to evaluating Emacs Lisp expressions but without contaminating current Emacs.

## Installation

Clone this repository, or install from MELPA. Add the following to your `.emacs`:

```elisp
(require 'elpl)
```

## Usage

```
M-x elpl
```
